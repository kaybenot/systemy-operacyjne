Napisane przez Kamila Butryna.

Kompilacja:
    Wpisać komendę `make all`.

Uruchamianie:
    Wpisać komendę `make run`.

Opis zadania:
    Program umożliwia obliczanie serwerowe przy pomocy komunikatów,
     można wydzielić program klienta i serwera gdzie komunikują się dzięki
     tym komunikatom. Program wydzielony jest na sekcję programów (src) i osobną
     bibliotekę (libcom). Możliwe są operacje + - * / na dwóch liczbach. Wystarczy
     wpisać liczby i znak między z odstępem spacji. Pliki wykonywalne lądują w 
     folderze build.
